# Operadores Aritméticos

> Realizam as operações básicas da matemática entre duas variáveis.
>
> Combinadas entre si, permitem a criação de expressões mais complexas.

| (+) |operador adição                |
|:---:|:-----------------------------:|
| (-) |operador subtração             |
| (*) |operador multiplicação         |
| (/) |operador divisão               |
| (%) |operador módulo (resto divisão)|
